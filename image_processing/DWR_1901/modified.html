Modified DWR_1901 sample imageÂ¶

The Daily Weather Report 1901 sample worked much better with the image contrast reduced to the point where humans can hardly read it.

Daily Weather Report 1901 sample image, after image adjustment, overlain by the text blocks produced by Textract.


Script to make the figure:

#!/usr/bin/env bash

# Tweak the contrast on the DWR 1901 sample to get 
#   much better Textract results.

../scripts/modify.py --source=../../../samples/DWR_1901_03_left.jpg \
                     --colour=1.0 \
                     --sharpness=1.0 \
                     --brightness=1.0 \
                     --contrast=0.1
../scripts/run_textract.py
../scripts/oplot_text.py

Using:

  • Script to adjust contrast, colour, brightness, and sharpness of image
  • Script to run Textract
  • Script to overplot extracted text on modified image
